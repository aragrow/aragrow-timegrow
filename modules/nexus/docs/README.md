## Plugin Architecture

The TimeGrow ecosystem consists of two main plugins:
1. **aragrow-timegrow** - Base time tracking functionality
2. **aragrow-timegrow-nexus** - Advanced features and integrations

Both plugins share a unified design system documented in `CSS-STANDARDIZATION.md`.
